gSewa(Greedy Sewa)

gSewa is a Flask program to analyze transaction/data from eSewa excel datasheet.

Requirements

  • First install Python 3.3 or newer version
  • Install requirements via pip
  pip install -r requirements.txt

Note:

Only few features works currently but don't worry all features will work soon :) Features not working:

  • Uploading of excel datasheet
  • Total Tab

Usage

Download excel datasheet from eSewa and put it in project directory Then initiate the database

python manage.py init_db <excel document>

For ex:

python manage.py init_db esewa.xls #if your excel file is named esewa.xls

Then runserver:

python manage.py runserver

Navigate to 127.0.0.1:5000/payment/all
